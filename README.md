# DashBoard-With-Ts
This is a sales dashboard project built with React and TypeScript, developed to improve my skills in frontend development and data visualization with interactive charts. The project was inspired by the Origamid course, which teaches how to build dashboards connected to sales APIs.

## Technology Used

- Typescript

- React

- Recharts

- CSS

- HTML

## Features

- Sales preview

- Select a date to view sales for that specifc date

- Select the month to view that month's sales (up to 3 months)

- Visualizing sales through graphs

- Access details of individual sales

## How to run the project

1. Step one: Clone the repository

```bash
    git clone "link"
```

2. Step two: Go to dashboard folder

```bash
    cd dashboard
```

3. Step three: Install the dependencies 

``` bash
    npm install
```

4. Step four: Start the development server:

```bash
    npm run dev
```

5. Step five: Open in your browser the link

[http://localhost:5173](http://localhost:5173)
